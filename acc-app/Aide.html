Comment créer un nouveau fichier Excel

Pour créer un nouveau fichier Excel, il faut cliquer sur le menu "Fichier" puis le sous-menu "Nouveau". Il faut ensuite définir son emplacement (généralement dans le dossier OneDrive) et son nom.

Attention, ce nouveau fichier sera protégé en écriture avec le mot de passe "aucoeurdeschats".

Exemple d'utilisation: Vous devez créer un nouveau fichier Excel pour démarrer une nouvelle année.

Comment ouvrir un fichier Excel existant

Pour ouvrir un fichier Excel existant, il faut cliquer sur le menu "Fichier" puis le sous-menu "Ouvrir". Il faut ensuite rechercher le fichier dans la boîte de dialogue (généralement dans le dossier OneDrive).

Exemple d'utilisation: Vous devez ouvrir un fichier Excel existant pour ajouter un élément au tableau des recettes ou des dépenses.
